## 1. 产品概述
民宿保洁交接 H5 应用——为保洁阿姨提供极简的手机端房间打扫交接工具，解决口头交接易遗漏、无凭据的问题。
- 目标用户：年龄偏大的保洁阿姨，操作习惯以点击为主，文字输入少
- 核心价值：3 步完成交接（选房 → 勾检 → 拍照），提交即留痕，民宿主后台可查

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 保洁阿姨 | 无需注册，直接访问 | 选房、勾选检查项、拍照、提交交接 |

### 2.2 功能模块
1. **交接页**：选房间号 → 勾选检查项 → 拍照上传 → 提交交接
2. **成功页**：提交成功反馈、查看交接记录

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能说明 |
|----------|----------|----------|
| 交接页 | 房间选择 | 网格式大按钮选择房间号，选即高亮，无需确认 |
| 交接页 | 检查项勾选 | 三个分组（床品/卫浴/厨房），每项大号复选框，一键全选 |
| 交接页 | 照片上传 | 大号拍照按钮，点击调起相机，最多 3 张，支持预览和删除 |
| 交接页 | 提交按钮 | 底部固定大按钮，未完成项置灰提示 |
| 成功页 | 成功反馈 | 大号勾号动画 + 文字提示，2 秒后可返回继续交接 |
| 成功页 | 交接记录 | 最近 10 条交接记录列表，含房间号、时间、缩略图 |

## 3. 核心流程
保洁阿姨打开 H5 → 选择刚打扫完的房间号 → 逐项勾选床品/卫浴/厨房检查项（可一键全选）→ 拍摄 3 张现场照片 → 点击「提交交接」→ 提交成功页显示 → 可继续下一间交接

## 4. 用户界面设计

### 4.1 设计风格
- 主色：温暖橙 #FF6B35（活力、亲和），辅色：柔和白 #FFF8F0（背景）
- 按钮风格：大圆角（16px），厚实 3D 感，按压有明显反馈
- 字体：系统圆体，标题 28px+，正文 20px+，复选框文字 22px+，确保阿姨看得清
- 布局：单列纵向，无侧边栏，底部固定提交按钮
- 图标：大号 emoji 风格图标，直观易懂
- 整体风格：温暖亲和、极简大字，减少认知负担

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 交接页 | 房间选择 | 网格布局（3 列），大号圆角卡片按钮，选中态橙底白字，字号 24px |
| 交接页 | 检查项勾选 | 分组卡片，每组带 emoji 标题，大号复选框（28px），每组顶部「全选」按钮 |
| 交接页 | 照片上传 | 3 个大号虚线框占位，点击调起相机，已拍照片缩略图预览，右上角删除按钮 |
| 交接页 | 提交按钮 | 底部固定，全宽圆角大按钮，未完成置灰，完成态橙底白字 |
| 成功页 | 成功反馈 | 居中大号绿色勾号动画，下方「交接成功」文字，再下方房间号和时间 |
| 成功页 | 交接记录 | 简洁卡片列表，含房间号、时间、缩略图 |

### 4.3 响应式设计
- 移动优先（H5 页面，主要在手机微信/浏览器使用）
- 适配 375px–428px 宽度手机屏幕
- 触控优化：所有可点击区域最小 48px × 48px
- 竖屏布局，无需横屏支持

### 4.4 3D 场景指引
- 不适用
