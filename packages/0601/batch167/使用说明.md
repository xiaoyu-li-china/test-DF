# Unity 可拖拽 UI 面板使用说明

## 功能特性
- 标题栏拖拽移动面板
- 右下角拖拽缩放面板
- 关闭、最大化、最小化按钮
- 最大化时面板填满父容器（动画过渡）
- 最小化时只显示标题栏（动画过渡）
- 恢复时还原之前的位置和大小
- 边界限制（不会拖出屏幕）
- 最小尺寸限制（默认 200x150）

## 使用步骤

### 1. 创建 UI 结构

在 Canvas 下创建以下层级结构：

```
Panel (添加 DraggablePanel 脚本)
├── Header (Image)
│   ├── TitleText (Text - 可选)
│   ├── MinimizeButton (Button)
│   ├── MaximizeButton (Button)
│   └── CloseButton (Button)
├── Content (Image/Panel - 面板内容区域)
├── ResizeHandle (Image - 右下角)
```

### 2. 详细设置

#### Panel (主面板)
- 添加 `DraggablePanel.cs` 脚本
- 设置锚点为 Center (0.5, 0.5)
- 设置 Pivot 为 (0.5, 0.5)
- 设置初始大小（建议 400x300）

#### Header (标题栏)
- 作为 Panel 的子对象
- 锚点：Top + Stretch
- Pivot：(0.5, 1)
- 高度：约 30-40
- 可以添加一个 Text 子对象显示标题

#### Content (内容区域)
- 作为 Panel 的子对象
- 建议添加 CanvasGroup 组件（最小化/最大化动画中可实现淡入淡出效果）

#### ResizeHandle (缩放手柄)
- 作为 Panel 的子对象
- 锚点：Bottom Right
- Pivot：(1, 0)
- 大小：约 20x20
- 位置：(0, 0)
- 颜色：半透明或显示一个缩放图标

#### CloseButton / MaximizeButton / MinimizeButton (标题栏按钮)
- 作为 Header 的子对象
- 锚点：Top Right（依次排列）
- 大小：约 25x25
- 可以设置为 "X"、"□"、"—" 图标
- 排列示例：CloseButton (-5, -5)，MaximizeButton (-35, -5)，MinimizeButton (-65, -5)

### 3. 脚本参数设置

在 Inspector 面板中为 DraggablePanel 赋值：
- **Header**: 拖拽标题栏的 RectTransform
- **Resize Handle**: 右下角缩放手柄的 RectTransform
- **Close Button**: 关闭按钮的 Button 组件
- **Maximize Button**: 最大化按钮的 Button 组件
- **Minimize Button**: 最小化按钮的 Button 组件
- **Content**: 面板内容区域的 RectTransform（缩放时自动重建布局，动画中控制显隐和透明度）
- **Min Size**: 最小尺寸（默认 200x150）
- **Header Height**: 标题栏高度（默认 36，用于最小化时的目标高度）
- **Animation Duration**: 动画过渡时长（默认 0.2 秒）

### 4. 交互行为

| 操作 | 效果 |
|---|---|
| 点击最大化按钮 | 面板动画填满 Canvas |
| 最大化状态点击最大化按钮 | 面板动画还原到之前的位置和大小 |
| 最大化状态拖拽标题栏 | 立即还原面板，鼠标位置对应还原后面板内的相对位置（类似 Windows 行为） |
| 点击最小化按钮 | 面板动画收缩为仅标题栏 |
| 最小化状态点击最小化按钮 | 面板动画还原到之前的位置和大小 |
| 最小化状态拖拽标题栏 | 立即还原面板，可继续拖拽 |
| 动画进行中 | 拖拽和按钮操作被忽略 |

## 公共 API

```csharp
// 关闭面板
panel.ClosePanel();

// 打开面板
panel.OpenPanel();

// 切换最大化
panel.ToggleMaximize();

// 切换最小化
panel.ToggleMinimize();

// 修改最小尺寸
panel.minSize = new Vector2(300, 200);

// 修改动画时长
panel.animationDuration = 0.3f;
```

## 注意事项

1. 确保 Canvas 的 Render Mode 为 Screen Space - Overlay 或 Screen Space - Camera
2. 如果使用 Screen Space - Camera，确保 Event Camera 已设置
3. 面板的 Pivot 建议设置为 (0.5, 0.5) 以获得最佳效果
4. 确保场景中有 EventSystem 对象
5. 如需最小化动画中内容区域有淡入淡出效果，请为 Content 对象添加 CanvasGroup 组件
6. headerHeight 需要与实际标题栏高度一致，否则最小化后面板高度不准确
