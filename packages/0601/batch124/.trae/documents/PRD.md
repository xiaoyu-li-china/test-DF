## 1. 产品概述
面向钓鱼俱乐部的潮汐预报嵌入小组件，用户选择浙江近海海域观测点后，可查看今日及未来3天的高潮/低潮时间与潮高/潮流曲线，系统结合潮况与月相给出适宜钓鱼指数（1-5星），支持收藏常去钓点，辅助钓友合理安排出钓时间。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览海域选择、查看潮汐预报 |

### 2.2 功能模块
1. **潮汐预报页**：海域选择器、收藏钓点、日期切换标签、适宜钓鱼指数、高低潮时间表、潮高/潮流曲线图

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 潮汐预报页 | 海域选择器 | 下拉选择浙江近海7个观测点（嵊泗、舟山、宁波、象山、台州、洞头、温州），选中后刷新潮汐数据；收藏的钓点置顶并带星标 |
| 潮汐预报页 | 收藏钓点 | 点击星标图标收藏/取消收藏常去钓点，收藏数据存入 localStorage，收藏的钓点在下拉列表中置顶显示 |
| 潮汐预报页 | 日期标签 | 今日/明日/后日/大后日 4个标签页切换，高亮当前选中日期 |
| 潮汐预报页 | 适宜钓鱼指数 | 结合当日潮况（潮差大小、涨落时机）和月相（新月/满月大潮优），给出1-5星评分，1-2星红色差、3星黄色中、4-5星绿色优，附简短理由文字 |
| 潮汐预报页 | 高低潮时间表 | 以卡片列表展示当日所有高/低潮时刻及对应潮高值，高潮用上升箭头标识，低潮用下降箭头标识 |
| 潮汐预报页 | 潮汐曲线图 | 支持切换「潮高」和「潮流速度」两种模式，以平滑曲线展示完整 0:00-24:00 变化趋势（不得截断凌晨或深夜数据），标注极值点位，纵轴带上下留白、横轴完整覆盖 0-24 时 |

## 3. 核心流程
用户打开小组件 → 从下拉菜单选择海域（或点击收藏钓点快速切换）→ 页面加载该海域潮汐数据 → 查看适宜钓鱼指数 → 默认展示今日潮汐 → 查看高低潮时间表和潮高/潮流曲线 → 切换曲线模式（潮高/潮流速度）→ 切换日期标签查看未来3天潮汐 → 收藏常去钓点 → 根据信息决定出钓时间

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：海洋深蓝风格，契合钓鱼/航海场景
- **主色**：深海蓝 #0B3D6B，搭配潮汐青 #00B4D8 作为强调色
- **辅色**：沙滩米 #F5E6CA 用于卡片背景，珊瑚橙 #FF6B35 用于高潮高亮
- **按钮样式**：圆角胶囊形，hover 时微发光效果
- **字体**：标题用 Noto Serif SC 衬线体营造质感，正文用 Noto Sans SC
- **布局风格**：纵向卡片式布局，顶部海域选择器，中间日期标签，下方潮汐信息区
- **图标风格**：线性海洋图标（波浪、锚、鱼钩等）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 潮汐预报页 | 海域选择器 | 下拉框，深蓝背景白色文字，选中项带波浪下划线动画；收藏钓点置顶并带实心星标 |
| 潮汐预报页 | 收藏按钮 | 海域名右侧空心/实心星标，点击切换收藏状态，带缩放动画反馈 |
| 潮汐预报页 | 日期标签 | 横向胶囊标签组，选中态带潮汐青底色和微光效果 |
| 潮汐预报页 | 适宜钓鱼指数 | 星级评分卡片，1-2星红色、3星琥珀黄、4-5星翡翠绿，下方灰色小字简述理由（如"大潮日，潮差大，鱼口活跃"） |
| 潮汐预报页 | 高低潮时间表 | 纵向卡片列表，高潮卡左侧珊瑚橙边框，低潮卡左侧潮汐青边框，内含时间和潮高值 |
| 潮汐预报页 | 潮汐曲线图 | 曲线区域右上角胶囊切换按钮「潮高」/「潮流」，潮高模式深海蓝渐变填充，潮流模式潮汐青渐变填充；高/低潮点标注圆点，网格线淡显 |

### 4.3 响应式
- Desktop-first 设计，小组件宽度 480px 固定（嵌入场景），高度自适应
- 移动端自适应全宽，潮高曲线改为触摸交互

### 4.4 3D场景指导
- 不适用
